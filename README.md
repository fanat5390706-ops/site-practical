## 📄 Описание проекта

Одностраничный сайт (Landing Page) вымышленной дизайн-студии **PixelForge** — выполнен в рамках практической работы в колледже. Сайт включает полный набор типовых секций современного коммерческого лендинга: шапку с поиском, навигацию, слайдер-герой, блок услуг, видеосекцию, отзыв, социальную панель и подвал.

## 📑 Секции сайта

### 🔝 Header (Шапка)
Залипающая (sticky) шапка с логотипом **Pixel`Forge`** и строкой поиска. Логотип набран шрифтом *Bebas Neue*, акцентная часть выделена лаймовым цветом.

### 🧭 Navigation (Навигация)
Горизонтальная навигационная панель на тёмном фоне с плавными якорными ссылками на все секции страницы: Главная, Услуги, О студии, Отзывы, Контакты.

### 🎠 Hero / Slider (Слайдер)
CSS-слайдер без JavaScript — управление через скрытые радиокнопки (`<input type="radio">`). Три слайда с разными заголовками и изображениями в формате **AVIF**. Внизу секции — SVG-волна для плавного перехода к следующему блоку. Точки-переключатели расположены под слайдером.

### 🛠️ Services (Услуги)
Три карточки услуг — **Веб-дизайн**, **Брендинг**, **Разработка**. В каждой карточке: иконка (`.avif`), заголовок, краткое описание и ссылка «Подробнее». Иконки хранятся в папке `images/`.

### 🎬 About / Video (О студии)
Двухколоночный блок: слева встроенное видео (`видео.mp4`) с атрибутами `autoplay`, `muted`, `loop`; справа — список из 6 пунктов, описывающих этапы работы.

### 💬 Testimonial (Отзыв)
Секция с цитатой клиента, именем и должностью — простой акцентный блок для повышения доверия.

### 📱 Social Bar (Соцсети)
Горизонтальная панель со ссылками на Facebook, Telegram, Instagram, YouTube.

### 🦶 Footer (Подвал)
Четырёхколоночная сетка (CSS Grid): Услуги, Проекты, Решения, Партнёры. Внизу — повторная навигация и копирайт.

## 🚀 Запуск

Никаких зависимостей и сборщиков не требуется. Достаточно открыть `index.html` в браузере:

```bash
# Вариант 1 — просто открыть файл
Открыть index.html двойным кликом в проводнике

# Вариант 2 — через Live Server (VS Code)
Нажать правой кнопкой на index.html → "Open with Live Server"

# Вариант 3 — через Python
python -m http.server 8000
# Затем перейти на http://localhost:8000
```

## 📁 Форматы медиафайлов

| Файл | Формат | Особенности |
|---|---|---|
| `слайдер1.avif` | AVIF | Современный формат, ~62 КБ |
| `слайдер2.avif` | AVIF | ~45 КБ |
| `слайдер3.avif` | AVIF | ~20 КБ |
| `слайдер1.jpg` | JPG | Запасной постер, ~23 КБ |
| `иконка1.avif` | AVIF | ~13 КБ |
| `иконка2.avif` | AVIF | ~45 КБ |
| `иконка3.avif` | AVIF | ~54 КБ |
| `видео.mp4` | MP4 | ~61 МБ, автовоспроизведение без звука |

> **Примечание:** формат AVIF обеспечивает значительно меньший вес файлов по сравнению с JPG/PNG при сопоставимом качестве, однако поддерживается не всеми старыми браузерами.
